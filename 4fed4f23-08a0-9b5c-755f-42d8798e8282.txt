　　萧逸伸出手，缓缓靠近这团魂力。
<br/>　　无论这团魂力是否是幻光洞府内的奥秘，或者说秘密所在。
<br/>　　其本身就是不可多得的重宝。
<br/>　　而且，萧逸此刻明显感觉到，体内的控火兽武魂，一阵跃动。
<br/>　　整团魂力，散发着氤氲的斑斓光芒。
<br/>　　光芒，照耀了整个主府。
<br/>　　单单是这些光芒，已让得控火兽武魂有此异动。
<br/>　　但让萧逸感到奇怪的是，往日见了精纯能量，便雀跃不已的冰鸾剑武魂，此刻却是安静无比。
<br/>　　嗤…
<br/>　　萧逸的手，搭在了这团魂力之上。
<br/>　　然而，手掌上却仿佛触之无物，搭在了一团空气之上。
<br/>　　手掌，亦直接穿入了这团魂力之中。
<br/>　　“嗯？”萧逸微微疑惑。
<br/>　　手掌传入这团魂力之中，并未有什么异况发生。
<br/>　　“怜星姑娘，你体内武魂，可有什么感觉？”萧逸回头看向顾怜星。
<br/>　　顾怜星面露异色，道，“我的紫极星辰武魂，似乎有些异动。”
<br/>　　“哦？”萧逸面带疑惑，随后认真打量了顾怜星一眼。
<br/>　　明显可以看到，顾怜星原本苍白的脸色，此刻在璀璨光芒的照耀下，变得红润了些。
<br/>　　“这团魂力之内的力量，应当对你的武魂有效果。”萧逸说了一声。
<br/>　　下一秒，手指轻动。
<br/>　　指尖缓缓抽出，一缕氤氲幻光，竟随着他的指尖缓缓飘荡而出。
<br/>　　“可以驾驭而出。”萧逸面露喜意。
<br/>　　既然面前这个色彩斑斓的圆球，乃是一团能量。
<br/>　　那便存在能否驾驭的问题。
<br/>　　而很显然，现今这团魂力力量，可以驾驭。
<br/>　　“祭出你的武魂。”萧逸吩咐一声。
<br/>　　“是。”顾怜星点点头，双眼微微闭起。
<br/>　　半晌，一团紫色的星云，凭空而现。
<br/>　　星云，宛若一个紫色的星光漩涡，围绕着顾怜星缓缓旋转。
<br/>　　这便是紫极星辰武魂的本体。
<br/>　　不过，此刻星光，异常黯淡。
<br/>　　证明着，顾怜星体内的武魂，已然虚弱无比。
<br/>　　“去。”萧逸低喝一声。
<br/>　　指尖游走，缕缕氤氲幻光，霎时打入紫色星云之内。
<br/>　　几乎是肉眼可见的速度，氤氲幻光瞬间消失，而紫色星云的光芒，则恢复了一些。
<br/>　　顾怜星脸色一喜，道，“我的武魂瞬间吸收了那缕魂力。”
<br/>　　“武魂明显被修补了一些。”
<br/>　　萧逸点了点头，“看来这些精纯魂力可以修补你的武魂。”
<br/>　　“开始吧，你专心操控好武魂即可。”
<br/>　　说罢，萧逸手指快速点动。
<br/>　　面前‘圆球’之内，不断被抽出缕缕氤氲幻光，那些便是圆球之内的精纯魂力。
<br/>　　缕缕精纯魂力，快速打入紫色星云之内。
<br/>　　紫色星云，亦快速吸收着这些魂力，以修补自身。
<br/>　　顾怜星连忙闭目，认真地操控着自己的武魂。
<br/>　　一旁，公孙火舞见状，面露大喜之色。
<br/>　　“太好了，怜星你的武魂开始在恢复了。”
<br/>　　顾怜星仍旧闭着目，面露喜意地点点头。
<br/>　　二人脸色欣喜。
<br/>　　萧逸却是边抽调着这些精纯魂力，边脸色凝重地打量着四周。
<br/>　　这个‘圆球’，里头魂力如此精纯浑厚，乃是不可多得的重宝之一。
<br/>　　这幻光洞府之内，若说没有危险，萧逸打死也不信。
<br/>　　虽然现在看起来没有异状，但谁知道之后会发生什么。
<br/>　　这些禁地之内的危险，萧逸亲身经历过，自然也无比清楚。
<br/>　　警惕之下，若真有异况发生，他也能瞬间带二人离开。
<br/>　　还是那句话，要抵御这里未知的危机，萧逸没有丝毫把握。
<br/>　　但若说逃遁，他倒是有把握。
<br/>　　十八府险地之内的那些禁地，其危险程度应该差不多；就算有差别，也差之不远。
<br/>　　金光禁地的金光风暴，还有天星禁地的星光风暴，他都见识过了。
<br/>　　确实无法抗衡，但逃遁保命还是能做到的。
<br/>　　而且，之前在金光风暴和星光风暴中逃命，都是深处地底之下，至于上天一途。
<br/>　　而现在，是地面之上。
<br/>　　只要他反应足够快，能瞬间出这洞府的话，那么便是天高地阔，四面八方任他逃遁，而至于只有上天御空一途。
<br/>　　故从这一点看来，即便这幻光洞府之内真有危险。
<br/>　　萧逸要逃遁的把握，都比之前在星光风暴或者金光风暴中要高许多。
<br/>　　这也是萧逸敢带顾怜星二人进来的最重要原因。
<br/>　　逃遁保命，他有足够的把握。
<br/>　　十数分钟后。
<br/>　　散发着璀璨光芒的‘圆球’之上，精纯浑厚元力不断被抽调而出。
<br/>　　顾怜星的紫极星辰，原本黯淡的光芒，亦不断变得光亮。
<br/>　　这代表着，她的武魂，正不断被修复着。
<br/>　　这时，萧逸微微分出一丝心神，注意着这些魂力修补紫极星辰武魂的情况。
<br/>　　半晌，他便恍然了。
<br/>　　这些精纯魂力，能够修补武魂，其原因，无他。
<br/>　　魂师的修炼，更准确来说，本就是武魂的修炼。
<br/>　　魂力，也是由武魂修炼而出。
<br/>　　魂力于武魂，就如元力与武者。
<br/>　　精纯的元力，能够替武者的身体本身治愈伤势。
<br/>　　精纯的魂力，自然也能够治愈武魂。
<br/>　　顾怜星的武魂，乃是紫色阶品的紫极星辰，而且还是极其稀有的武魂类别。
<br/>　　故正常的魂力，或者说一般的魂力，即便有治愈效果，效果也不大，甚至微不可闻。
<br/>　　可这个‘圆球’内的魂力，精纯无比，自然是效果极大。
<br/>　　半个时辰后。
<br/>　　紫极星辰武魂上的光芒，耀眼到了极点。
<br/>　　顾怜星霎时收回了武魂，深深呼了口气。
<br/>　　“武魂修补完了吧。”萧逸问道。
<br/>　　顾怜星点了点头，喜悦道，“这么多年了，除了幼时刚刚觉醒武魂时外，现在还是头一次武魂光芒如此耀眼。”
<br/>　　“那便好，走吧。”萧逸点了点头。
<br/>　　顾怜星的武魂修复好，那么来幻光险地的目的，便算完成了。
<br/>　　再观这个‘圆球’，几乎没什么变化。
<br/>　　被抽调了如此多的精纯魂力，去修补一个即将溃散的紫色武魂，却几乎不见有明显的能量消耗。
<br/>　　不难想象，这个‘圆球’内蕴含的魂力数量，异常惊人。
<br/>　　不过，刚才萧逸不断抽调出精纯魂力时也发现，他越是抽调的数量多，他的驾驭程度就越加困难。
<br/>　　到了此刻，他驾驭的难度已极大。
<br/>　　“出。”萧逸低喝一声，再次抽调了一丝精纯魂力。
<br/>　　这一次，他将其吸入了体内。
