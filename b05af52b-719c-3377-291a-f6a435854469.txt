　　第402章
<br/>　　和珅很快来到书房，而阿喜和白元飞已经离开。白元飞执行这次任务，不仅冒了很大风险，也付出极大的辛苦，如今能平安回来，的确需要好好休养一阵。
<br/>　　“大当家，你找我什么事？”和珅。
<br/>　　“与凉州城谈判，用宋游渠交换福德勒。”
<br/>　　程大雷劈头盖脸甩过来一句话，和珅还不知怎么会事，此刻是一头雾水。
<br/>　　“出什么事了？”
<br/>　　程大雷简单将前因后果说了，最后道：“与凉州城谈判的事情，就交给你负责，尽快把这件事谈妥，不能耽搁。”
<br/>　　时间耽搁得越长，福德勒就越危险，所以程大雷需要在最短时间把这件事解决。
<br/>　　“这件事交给你没问题吧？”程大雷问。
<br/>　　“自然没问题。”和珅心里琢磨着，宋伯康定然不会为了个无足轻重的福德勒，舍下自己的儿子，所以这次谈判己方还是有优势的：“只是，大当家真要舍了宋游渠么，他在我们手中，可是一个重要的筹码。”
<br/>　　“所谓筹码，就是拿起来用的，否则压在手中做什么，炫富么？”
<br/>　　这次和珅没有奉承程大雷的话，而是道：“可是武器只有没出手前，才最有威胁？”
<br/>　　“呃……”程大雷当真被噎得无话可说，无奈的挥挥手：“去做吧，其他我都考虑清楚了。”
<br/>　　和珅立刻派人联系凉州城，希望按照程大雷的嘱咐，最短时间内完成这件事。
<br/>　　而程大雷自己则特意去看望了宋游渠一伙人。
<br/>　　自从被扣下来做人质后，宋游渠一伙纨绔子弟就住在琴川关。倒是没有被软禁，程大雷可没有余粮养闲人。甚至某些人还在琴川关担任了一定职务，每月都有薪酬可以拿。
<br/>　　屈指算来，他们来到琴川关，已经一年有余。毁灭一个人有两种方式，一种是精神上的，一种是身体上的。
<br/>　　杀死一个人或许只需要一刹那的拔刀，而毁灭一个的精神需要多长时间？
<br/>　　应该用不了一年时间这么长。
<br/>　　程大雷对宋游渠的精神洗脑从未停止过，这是在山寨他罕见亲手抓的事情之一。
<br/>　　从早到晚，都执行军事化管理，重复繁重的劳动，可以麻痹他们的神经，然后不停灌输的口号，足以令他们接受一些看起来正确，但其实无比荒谬的道理。
<br/>　　程大雷出现的时候，宋游渠正在誊书，将书一本本抄下去，发到学堂作为教材使用。
<br/>　　看到程大雷出现，宋游渠十几人同时站起来，用崇拜的眼神注视着程大雷。
<br/>　　“坐，坐，大家不要这么紧张，我就是随便过来看看。”程大雷满脸和蔼的笑，如春风化雨：“我今天过来，就是想和大家聊聊。”
<br/>　　都说江山易改本性难移，或许骨子里的东西不是那么容易改变的。但精神上的控制，其实并不困难，人在进入集体后，会下意识屈服集体所制定的规则，哪怕在外人看来，这种规则傻到极限。
<br/>　　看这些人高昂的精气神，忐忑紧张的表情，这证明程大雷的洗脑工作做得不错。程大雷盘腿在案前坐下，抬起两手向下压了压，气氛一片其乐融融。
<br/>　　“诸位，长安城的事情你们可听说了？”
<br/>　　程大雷脸上带着笑意，却眼神如电，观察着这些人的一举一动。他分明看见，自己的话出口，周围人的眼神都炙热起来，表情隐隐有些亢奋。
<br/>　　长安城闹得沸沸扬扬，这些人虽然了解得不甚详细，却也有个耳闻。只听说，天下诸侯与绿林枭雄齐聚长安城，一场将决定帝国未来的战场正如火如荼的进行。
<br/>　　他们是纨绔子弟不假，但他们同样是年轻人，体内的血同样炙热。热血男儿，谁不想轰轰烈烈的做一番事业，青史留名也好，一代枭雄也罢，谁不愿在史书上留下浓墨重彩的一笔。
<br/>　　“大当家，要出兵么？”宋游渠问。
<br/>　　程大雷笑了笑，没有回答，反而问道：“你们对这一战如何看？”
<br/>　　随之，屋内的气氛便热切起来，大家七嘴八舌，争论得十分火热。
<br/>　　程大雷笑看着这一幕，同时不得不承认，他们的话里并非一点道理没有。这些人虽然被程大雷集体化管理，无时无刻的精神洗脑，但这并不代表他们是傻瓜。
<br/>　　一匹烈马被蒙住眼睛，并不代表它无法奔驰，只是分不清方向而已。
<br/>　　“大武气数已尽，即便能撑过这一战，也是时日无多。其后，必然是诸侯林立，群雄逐鹿，直到有人可以一统神州，重定天下。”
<br/>　　“很好。”程大雷赞许的点点头：“你觉得谁可以得天下？”
<br/>　　“这……”宋游渠无奈的摇摇头：“天下大势，瞬息万变，宋某才疏学浅，现在看不出来。”
<br/>　　程大雷哈哈笑了一阵，笑声有些阴冷。
<br/>　　“未来无论谁能得天下，必然是刀兵连绵，生灵涂炭，苦得也只能是天下百姓而已。诸位……”程大雷拖了个长音：“诸位都是大好男儿，生逢这乱世，自然要靠手中刀，怀中笔立一番工业。天下大势难断，诸位对自己的前程可有安排？”
<br/>　　一句话点燃了诸人的心头火，对宋伯康这些人来说，目的是做个战争投机分子，让自己的家族可以传承千代万代。
<br/>　　然而，对于宋伯康这些年轻人来说，家族传承，富贵荣华这些重要么？
<br/>　　重要，也不重要。和理想、信仰、改变世界这些美好的字眼相比，那些富贵什么都不是。
<br/>　　对于年轻人来说，只要你给他们一点火，他们就会毫不犹豫的燃尽自己，哪怕粉骨碎身。
<br/>　　程大雷亮起了火把。
<br/>　　程大雷脸上的微笑缓缓收敛。他直起身首领光环开到极致，在诸人的眼睛中，程大雷的形象变得无比伟岸，身上若有光芒绽放。
<br/>　　“我曾经有一个理想，也曾捧过圣贤书，握过青锋剑，也曾想习得文武艺，货与帝王家，也曾说男儿何不带吴钩，收取关山五十州……”
<br/>　　程大雷缓缓向诸人探出手。
<br/>　　“现在我这双手，只想握住一名同伴的手，诸位……”程大雷用深沉磁性的声音道：“愿意么？”
<br/>(本章完)
