<br/>    现在陈明也就能算个大概，至于更多的，就算不出来了。
<br/>    
<br/>        而且，自己要在这一个应了星辰的仙人附近，才能看出来。
<br/>    
<br/>        估摸着，《皇极经天道》参悟的越多，能够看到的，也就越多，也就更加清楚。
<br/>    
<br/>        带着凌仙和李素衣，还有不少参与了战役的记名弟子，陈明一行人来到了宝玄寺外。
<br/>    
<br/>        宝玄寺的山门之处，二十多个地仙静静站立，等候着陈明到来。
<br/>    
<br/>        不说陈明帮助他们度过了兽潮，就说如今燕山的实力，吞并他们跟玩一样，就当是拜山头，那也得来啊。
<br/>    
<br/>        燕山，在这一带，怎么也算是一条地头蛇，他们自然是要前来拜会。
<br/>    
<br/>        为了表示诚意，所有地仙都在门口静静等待。
<br/>    
<br/>        见到陈明落下，众多地仙齐齐道:“恭迎燕山山主前来赴宴！”
<br/>    
<br/>        道完，众多的地仙，以及他们门下的弟子，都是齐齐弯腰行礼。
<br/>    
<br/>        陈明微微点头，“大家不用多礼，准备赴宴吧。”
<br/>    
<br/>        陈明跟着丰禄方丈，走进偏殿之中，准备参加宴会，至于说为啥是偏殿，不是正殿，这并不是不尊敬陈明，而是人家正殿供奉的是佛祖啊。
<br/>    
<br/>        来到偏殿之中，丰禄方丈笑盈盈地拱手道:“请山主先进！”
<br/>    
<br/>        陈明笑道:“这怎么好意思？”
<br/>    
<br/>        丰禄方丈道:“这一次能够抵抗住兽潮，全靠燕山上下齐心协力，捍卫天下正道，若不是英明神武，心怀天下的山主，倾覆之下，安有完卵？山主先进，我等才敢进去啊。”
<br/>    
<br/>        其他仙人也是道:“如今仙路凋敝，人心不古，像是山主这般乐于助人的仙人，可是不多了，还是山主先进！”
<br/>    
<br/>        陈明扫了周围一眼，这些地仙都是面露敬仰之色，而那些小辈，特别是女孩，已经是两眼放光了。
<br/>    
<br/>        自己现在有这么大牌面吗？
<br/>    
<br/>        不过陈明也没有纠结，既然大家这么给面子，也不能拒绝大家的好意啊。
<br/>    
<br/>        陈明跨步，第一走进大殿之中，却是看到，其中有着一张张桌案，大概是十个人一桌。而在最高处，还有着单独的一张方桌。
<br/>    
<br/>        嗯，菜肴也还不错，看起来挺好吃的样子。
<br/>    
<br/>        就是十个人，不知道够不够吃啊。
<br/>    
<br/>        陈明正想随便找个位置坐下，却是见到丰禄方丈带着自己，来到最高处的桌案之前，双手合十道:“山主，请坐！”
<br/>    
<br/>        陈明心说，这还有特殊化的！
<br/>    
<br/>        陈明在正上首的位置坐下，丰禄又是朝着凌仙和李素衣拱手道:“二位请坐！”
<br/>    
<br/>        在陈明左右，还有着两个位置，便是凌仙和李素衣的位置。
<br/>    
<br/>        陈明心说，本来我还很高兴，一个人吃一桌，怎么你把这两个二货也安排在我的旁边了？
<br/>    
<br/>        不过陈明也不露声色，免得这两个二货以为自己厌恶他们。
<br/>    
<br/>        凌仙和李素衣在陈明的一左一右坐下。
<br/>    
<br/>        众多地仙这才落座，陈明大概看出来了，修为越高的，距离自己越近，而凌仙和李素衣，估摸着是因为这一次战争之中极为出彩，所以也很是受人尊敬，虽然只是超脱的修为，却是地位很高。
<br/>    
<br/>        当然，就算真的动手，这两个二货不是还有仙兽吗？
<br/>    
<br/>        也勉强算得上是仙人战力了。
<br/>    
<br/>        这个时候，一位地仙犹豫着站了起来，朝着陈明拱手问道:“听闻燕山有五位宗主，俱是人杰，为何此次只来了二位宗主，难道是我等招待不周吗？”
<br/>    
<br/>        众多仙人也是心有余悸，对于燕山的实力，敬畏不已，只是两位宗主，就已经横扫了规模如此之大的兽潮，燕山可是还有着三位宗主没有露面呢。
<br/>    
<br/>        陈明笑道:“无妨，我门下大弟子正在闭关，四弟子和五弟子有事，走不开，燕山毕竟有这么大的烂摊子，四弟子五弟子不在，我不放心。”
<br/>    
<br/>        这位地仙这才坐下，“原来如此。”
<br/>    
<br/>        陈明本想直接让凌仙下去坐的，实在是不想呆在的范围内啊！
<br/>    
<br/>        不过看在这一次凌仙劳苦功高，给他一个机会吧。
<br/>    
<br/>        凌仙第一次能够靠着陈明坐这么近，也是有些喜不自胜，而且师父似乎是没有嫌弃自己，连忙拿起筷子，给陈明夹了一颗灵果，“师父，这灵果入口芬芳，加上特制的蜜·汁腌制，味道是极好的，还有这果脯，入口即化，酸中有甜，柔韧爽口，最是不错！”
<br/>    
<br/>        陈明不置可否，默默地尝了尝，别说，味道是还不错啊，特别是这果脯，酸酸的，却又不涩口，味道是极好的。
<br/>    
<br/>        这个时候，众多的修行者也是齐齐起身，朝着陈明举杯道:“此次兽潮来临，全倚仗山主大恩，我等才得以保全，敬山主！”
<br/>    
<br/>        陈明本身就没什么架子，当即也是起身，举杯道:“饮胜！”
<br/>    
<br/>        一口饮尽杯中酒，又有人道:“此次战役，二宗主坐镇军中，所有修行者无不奋勇向前，二宗主，就如同那定海神针一般让我等安心，我等，敬二宗主！”
<br/>    
<br/>        凌仙起身道:“都是应该的，应该的！”
<br/>    
<br/>        而后，众人又是敬了李素衣，“三宗主此次劳苦功高，一路征伐数十颗星辰，扫荡兽潮，挽救苍生，我等，敬三宗主！”
<br/>    
<br/>        李素衣淡淡起身，不喜不悲，默默饮下。
<br/>    
<br/>        众人也是知道李素衣的脾气，不仅没有生气，反而是喜不自胜。
<br/>    
<br/>        终于给这个从来没有赴宴的三宗主敬酒了！
<br/>    
<br/>        陈明埋头对付着桌上的菜肴，至于敬酒什么的，这两个二货不是很能喝吗？让他们喝去。
<br/>    
<br/>        宴会之上，杯筹交错，都是兴高采烈，庆贺这一次的胜利。
<br/>    
<br/>        一个地仙，却是小心地来到了陈明的身前，拱手道:“山主修行向来是很清苦的，不才在下有两个女儿，乃是双胞胎，更是出落得亭亭玉立，风姿绰约，年方十八，她们仰慕山主的风姿，希望可以在山主身边做两个剑侍！”
<br/>    
<br/>        陈明扫了一眼这面前看起来已经有九十岁的老头，都这么大年纪了，还有两个十八岁的女儿，还真是老当益壮啊！
<br/>    
<br/>        陈明估计，应该就在那些看自己双眼放光的女孩之中，双胞胎，听起来很有诱惑力啊！
<br/>    
<br/>        不过陈明并没有去寻找这两个女孩，估摸着也就看上一眼。
<br/>    
<br/>        带回燕山，怕不是这两个女孩活的不耐烦了呦！
<br/>    
<br/>        看看整个燕山分部主山峰，除了卓青瑶看得到第二个女孩？
<br/>    
<br/>        而且卓青瑶还振振有词，凌仙和李素衣年纪还小，不懂事，未免道心不稳，沾染红尘，被美色所诱惑，所以燕山分部主山峰之上，没有一个女孩，当然，也没有一个男人。
<br/>    
<br/>        将这两个女孩带回去，只怕是活不过三天。
<br/>    
<br/>        但是，我陈明岂是怕徒弟的人？为了凌仙和李素衣的幸福，我陈明就豁出去了！
