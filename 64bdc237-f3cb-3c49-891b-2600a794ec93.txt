　　陨星山脉，位于紫云城南面百里之外。
<br/>　　路途遥远不说，城外的山地也十分颠簸崎岖，山林野地密集。
<br/>　　除了洞玄境以上的武者，能御空飞行，转瞬便到之外，其他武者，单靠双脚或者骑马赶路，至少需要一天以上的时间。
<br/>　　趁闲着，萧逸拿出了临行前三长老给的乾坤袋。
<br/>　　这是一个中级乾坤袋，内含三十立方左右的乾坤空间。
<br/>　　说起乾坤袋，萧逸不禁感到这个世界的神奇。强大的武者、武魂、武技、丹药等等，竟还拥有着乾坤袋这样方便的东西。
<br/>　　小小一个袋子，挂于腰间，即可帮助武者装很多东西，减轻了不少负担。
<br/>　　不过乾坤袋的制造很麻烦，材料也不简单，所以售价非常贵。
<br/>　　乾坤袋共分三个等级。
<br/>　　低级乾坤袋，只有五个立方的空间，售价百两。
<br/>　　中级乾坤袋，足有三十个立方，售价千两。
<br/>　　高级乾坤袋，足足一百立方，售价万两。但这东西市面上看不到，整个紫云城也就三大家族拥有。
<br/>　　寻常独行武者，一般带个低级乾坤袋就够了。
<br/>　　中级乾坤袋，一般是后天以上的武者，或者大家族的子弟才用得起。
<br/>　　萧逸打开了乾坤袋，大致看了眼，都是些外出行走必备的东西。疗伤用的丹药、充饥的干粮、换洗的衣物等等。
<br/>　　“咦？”萧逸忽然发现，乾坤袋内竟还有一大叠银票，足有万两。
<br/>　　“这么多钱？”萧逸心中一喜。
<br/>　　武者修炼，本身就是非常烧钱的行为。
<br/>　　单单丹药就非常耗钱。
<br/>　　这笔钱，足够萧逸买更多的材料，炼制更多的丹药给自己。
<br/>　　不知不觉间，萧逸已经离开了紫云城。
<br/>　　郊外野道之上，人迹罕至，除了两旁的密林之外，便只剩下骏马赶路的‘踏踏’声。
<br/>　　一阵微风吹过，两旁的林子摇曳作响。树下草丛随风摆动，似有无数人头攒动。
<br/>　　“嗯？”萧逸忽然皱起了眉头。
<br/>　　“不对劲。”萧逸发现，四周的环境极为静谧。
<br/>　　以萧逸敏锐的察觉力，立马发现了不对劲的来源。
<br/>　　风吹树摆，不闻飞鸟鸣叫；树摆草动，不见野兽行动。
<br/>　　这里虽然是郊外野道，没有人烟，但野兽飞鸟是必然存在的。树木草丛如此摇曳，正常情况下必然会惊到这些动物。
<br/>　　可现在连一声鸟啼，一只野兔都见不到。
<br/>　　这证明，这附近的鸟兽都被事先赶走了。
<br/>　　而会赶走鸟兽的，只有人。
<br/>　　“不好，有埋伏。”萧逸瞬间拉住了马缰，停了下来。
<br/>　　一股浓郁的危机感忽然油然而生。
<br/>　　嗖的一声，一支利箭从萧逸的前方一米处横穿而过。
<br/>　　幸好他及时拉住了马缰，否则刚才那支利箭就会射中他。
<br/>　　刷、刷、刷、、、
<br/>　　两旁密林中，快速跑出数十个手持兵器的大汉。
<br/>　　“大哥，这次逮着个肥羊了。看这少年衣着不凡，显然是个有钱的主儿。”
<br/>　　“啧啧，埋伏了这么久，终于能干一票大的了。”
<br/>　　大汉们丝毫没有掩饰自己的话语，满脸得意。
<br/>　　数十大汉前，有一身材魁梧的刀疤脸，举起手中长剑指向萧逸，道，“少年，我们只劫财，不要命。留下钱财和衣服，放你离去。”
<br/>　　刀疤脸狞笑道，“别逼我们动手，你那小身板，还不够我一刀砍的。”
<br/>　　“呵。”萧逸一阵淡笑，“原来是遇到山匪了。”
<br/>　　这些野道，存在着山匪很正常。专门打劫过路的商客或者过路的行人。
<br/>　　不过，这些山匪也就恃着人多势众罢了，每一个的修为都不高，一般都在凡境二重左右，为首的头领也不过是凡境四重。
<br/>　　当然了，这些山匪中不乏亡命之徒，性格狠辣，手上人命累累。
<br/>　　遇到他们心情好的话还算幸运，若是他们心情不好，必然会杀人越货，要财也要命。
<br/>　　“没想到刚出紫云城就要到打劫的。”萧逸玩味一笑。
<br/>　　以他的实力，收拾这种山匪轻而易举。
<br/>　　“嗯？”忽然，萧逸脸色一变。
<br/>　　“不对。”萧逸目光一凝，想到了某些不对劲的事情。
<br/>　　“区区山匪，怎么可能让我升起危机感。而且，那股浓郁的危机，显然是威胁到我，才会让我的直觉率先发现。”
<br/>　　“有别的人在埋伏。”萧逸心中瞬间明悟。
<br/>　　果然，萧逸心中的念头刚闪过，前方一道冰锥疾速打来。
<br/>　　“哼。”萧逸冷哼一声，一拳将那冰锥打破，散落漫天冰屑。
<br/>　　同一时间，三道人影破空而来，个个气息不凡。
<br/>　　“武者，是强大的武者。”刀疤脸修为太低，第一次见到这样强大的武者，立刻被这三人的气息吓得瑟瑟发抖。
<br/>　　其他山匪也是满脸惊恐。
<br/>　　三人来到后，看了眼这些山匪，眼中露出一股不屑和讥讽。
<br/>　　“一群蝼蚁般的杂碎，污了我的眼睛。”
<br/>　　话语刚落，其中一人轻轻地挥了挥手。霎那间，数十跟水箭从天而降。
<br/>　　嗖、嗖、嗖、、、
<br/>　　水箭看似柔弱，实则极为强大，轻松穿透了这些山匪的身体。
<br/>　　不出数秒，数十山匪毙命，死于这些水箭之下。
<br/>　　“碧波水吟箭，你们是慕容家的人。”萧逸目光顿时一冷。
<br/>　　碧波水吟箭，黄阶巅峰武技，乃是慕容家的成名武技之一。
<br/>　　“哦？小子，有些眼力。”三人中为首的一人挑了挑眉毛，道。
<br/>　　“没想到萧家有名的废物，倒没有想象中那么不堪。”
<br/>　　“不过，也就仅此罢了，废物始终是废物。”
<br/>　　其余两人看向萧逸的目光极为不屑，透漏着一丝嘲弄的意味。
<br/>　　“呵。”萧逸冷笑一声，“好狗不挡道，慕容家的，滚开吧。”
<br/>　　“滚？”为首的中年人冷笑道，“萧逸，污了我家小姐的清白，你还想有命活下去？”
<br/>　　“慕容娇儿？”萧逸先是问了一声，而后道，“孰是孰非，你我心知肚明。我有没有玷污她，你们慕容家的人也清楚得很。”
<br/>　　萧逸冷声道，“想杀我，动手便是，何必还说得如此冠冕堂皇。”
<br/>　　从慕容家的这三人出现，并将这些山匪立刻击杀时，萧逸就知道，这三人摆明了是想要自己的命。
<br/>　　他们杀掉那些山匪，就是想杀人灭口，免得事情传出去。
<br/>　　“萧逸，在我们三人面前，你还能镇定自若，不露惧色。不知说你有胆识好，还是愚蠢好。”
<br/>　　“跟这小废物说那么多干嘛，直接杀了就是。”
<br/>　　“杀我？”萧逸脸色自若，道，“就凭你们三个，还没有那个本事。”
<br/>　　“猖狂。”
<br/>　　“大言不惭。”
<br/>　　“萧逸，别以为你打败了萧若寒和萧若狂，就有资格狂妄。在我们面前，你这种小辈，我们一根手指头就能捏死。”
<br/>　　这三人面露怒色，当然，他们也有十足的把握轻松杀掉萧逸。
<br/>　　毕竟，他们三人都是慕容家的执事。为首中年人乃是后天四重，其余两人则是后天三重。
<br/>　　在他们心中，杀一个凡境九重，不过是吹口气那么简单。
<br/>　　“慕容华执事，一个小废物罢了，还没那个资格让您出手，让我去废了他的修为，让他跪在你面前叩头认错。”其中一个执事说道。
<br/>　　慕容华，正是为首的后天四重。
<br/>　　而刚才说话的这个只执事，乃是后天三重，瞬间出手，一拳轰向萧逸。
<br/>　　人未至，嘴角已经露出了狞笑。在他看来，自己的一拳，足以将萧逸轰成重伤。
<br/>　　萧逸怡然不惧，同样一拳轰出。
<br/>　　“小废物，敢跟我刚正面，你找死。”慕容家执事冷笑一声。
<br/>　　然而，当两拳对轰之时，慕容家执事的冷笑却瞬间定格。
<br/>　　因为，萧逸竟接住了他这一拳，毫发无损。
<br/>　　“怎么可能？一个凡境九重根本接不下我这一拳。”慕容家执事先是疑惑，随后一惊，道，“体内真气瞬间打出，小废物，原来你已经炼化了一条主脉，你不是凡境九重，而是后天一重。”
<br/>　　“哼，难怪你如此狂妄。不过，我未出全力罢了，这一拳，要你的命。”说罢，慕容家执事后天三重的修为全力打出。
