　　萧逸御空飞离。
<br/>　　高空中，花影城一众强者面面相觑，却不敢强留。
<br/>　　萧逸也没闲工夫理会，如今情报再断，他本就心情不好，自是对这些打算纠缠的人没有好语气。
<br/>　　再者，这些邪修分部平日躲在暗处，八殿武者查不到也就罢了。
<br/>　　现在邪修分部被他端了，已经在明面上了。
<br/>　　若是这里的分殿连追查些许蛛丝马迹，确定这里是否邪修分部的本事都没有，那他萧逸也懒得和他们废话。
<br/>　　离开花影城，萧逸并没有离开弄花地域。
<br/>　　而是前往弄花地域的中心大城，弄花城。
<br/>　　每个地域，都有主殿。
<br/>　　当然，这里的主殿，只是普通主殿。
<br/>　　萧逸去了趟猎妖殿弄花城主殿，查阅了一番情报。
<br/>　　果然，弄花地域这半月时间以来，有不少地域内的天骄神秘失踪。
<br/>　　其中还有好些圣皇境强者。
<br/>　　显然，这些事都是邪修所为。
<br/>　　难怪他昨日在花影城花家内，没见到多少邪修。
<br/>　　而一天后那些邪修陆续回来，却一个个手持武道冰晶，且里头已经装满了武道传承和武道力量。
<br/>　　如无意外，弄花地域这半月时间内失踪的天骄和强者，已经全部身死了。
<br/>　　武道冰晶这种邪物，只能吸收死去武者的武道传承和力量。
<br/>　　另外，如果没猜错的话，当日公孙火舞，肯定也是在弄花地域这边遇到的邪修。
<br/>　　恰好碰上了邪修在大肆击杀地域内的天骄强者。
<br/>　　这才有了公孙火舞被邪修一路追杀的事。
<br/>　　“这些邪修搜集那么多武道传承和武道力量做什么？”萧逸皱了皱眉。
<br/>　　就他这些天来，所遇到的邪修，还有邪修手段，大大超乎了他以往的想象。
<br/>　　邪君府的本事还有手段，绝非区区武道冰晶能够涵括的。
<br/>　　一种种莫测诡异的手段、能力，还有一种种莫名而强大的邪物，都表明着邪君府这个藏在暗处的庞然大物拥有着异常恐怖的底蕴和实力。
<br/>　　……
<br/>　　萧逸已离开弄花地域。
<br/>　　十天后，距离弄花地域十数地域之外的某地域之内。
<br/>　　嘭…
<br/>　　一股狂猛的火焰，在妖兽森林上空爆发。
<br/>　　点点火星落下，精准地落在了一头头妖兽身上。
<br/>　　火星，看似微弱，但在碰到妖兽的瞬间，陡然爆燃。
<br/>　　爆燃之火，呈现金色，却又如蛆附骨，无物可挡。
<br/>　　哪怕是圣皇境巅峰妖兽，都在这点点火星之下，化作灰烬。
<br/>　　待得高空中的火焰爆发消失。
<br/>　　大片险地之内的妖兽，已然死伤殆尽。
<br/>　　“呼。”萧逸深呼吸一口气。
<br/>　　与其说是深呼吸，倒不如说，他在压抑自己的愤怒情绪。
<br/>　　整整十天，他横跨十数地域，横扫了十数个辽阔险地。
<br/>　　邪修的情报断了，他也难以追查到别的分部。
<br/>　　只能像往日的正常历练般，行走于各大妖兽森林，各大险地。
<br/>　　他的移动方向，乃是往东边而走。
<br/>　　中域，确实很大，而他这些年历练过、闯荡过的险地和地域，也不少了。
<br/>　　倒是东边范围的地域，他还未踏足过。
<br/>　　故如果只能正常历练的话，他便沿着这个方向，一路历练了。
<br/>　　只是，即便身处历练途中，他仍旧难以压抑心头愤怒。
<br/>　　咔咔咔…
<br/>　　萧逸的拳头，握得劈啪作响。
<br/>　　周遭遍地妖兽尸骸，似也难抑他心头情绪。
<br/>　　“不杀光你们，我如何心安。”萧逸的眼眸，异常冰冷。
<br/>　　萧逸冰冷的目光，并非看向周遭妖兽，而是看向手中一块邪道阵盘。
<br/>　　事实上，他很少对一些事情上心。
<br/>　　他有着更重要的事，他来中域的两个目的，还未完成。
<br/>　　但，若是这些事，与他的两个目的有关，他自然便也上心了。
<br/>　　邪君府，即便再怎样强大，邪修，即便再怎样阴险狡猾，也与他无关。
<br/>　　天大地大，萧逸还未怕过什么。
<br/>　　但这些邪修，若将目标放到他在乎的人身上，那便是他不可能不在乎之事。
<br/>　　虽然不知道圣月宗到底有多强大。
<br/>　　但，再怎样强大的宗门，再怎样强悍的底蕴，其内天骄，终归都是需要出外历练的。
<br/>　　闭门造车，永远不可能成就一位强者。
<br/>　　她既然会出外历练一次，便肯定会有第二次、第三次…
<br/>　　而以萧逸的了解，这些邪修凶残至极，且阴险过人，行事又肆无忌惮。
<br/>　　能有第一次埋伏围攻，便也必有第二次、第三次…
<br/>　　“圣月宗，我不知道你是护不住我家侍女，还是有别的原因。”
<br/>　　“但不代表我认可此事，这笔账，我留着上圣月宗再与你们算。”
<br/>　　在毁掉那些邪修分部时，萧逸多少找到些当日一战的情况情报。
<br/>　　虽不详细，但也多少有记录。
<br/>　　其中一句‘她险些殒命’，是拨动萧逸心弦的一句。
<br/>　　闲散的邪修，或许会很低调。
<br/>　　但囊尽天下邪修的邪君府，却绝不低调。
<br/>　　弄花地域这边的天骄、强者一一失踪之事，还有其它地域的邪修祸患，就是证明。
<br/>　　邪修这般猖獗，若不杀到害怕，杀到他们不敢冒头，萧逸如何心安。
<br/>　　那一战，她能拼着重伤，从邪修围攻中逃离。
<br/>　　下一次呢？
<br/>　　他现今实力，还上不了圣月宗，但不代表他容许她被邪修盯上。
<br/>　　他或许也奈何不了邪君府，但不代表他眼中容得下这些邪修。
<br/>　　“猎妖殿没有你们的情报，我便回风刹殿总殿查，风刹殿查不到，我便逐个地域查，我总有办法逮到你们。”
<br/>　　萧逸的语气，冰冷到极点。
<br/>　　手中握着的邪道阵盘，捏得极紧。
<br/>　　“嗯？”忽然，萧逸似是想到了什么，双眼忽然一眯。
<br/>　　他想起来，他曾经判断过，邪修，不是一个称呼，而是一门职业。
<br/>　　“若邪修是一门职业，有着他们所修之道的话，或许…”
<br/>　　他忽然想到了什么。
<br/>　　嗡…
<br/>　　空气中，一个无形漩涡，凭空而现。
<br/>　　萧逸的眼睛，蓦地变了，太阴太阳之眼，陡然凝聚。
<br/>　　他的眼睛，看向了手中邪道阵盘。
<br/>　　这邪道阵盘，里头力量，诡异莫测。
<br/>　　但此刻，在他眼中，这些诡异力量，忽然变得再简单不过。
<br/>　　萧逸的目光，又看向了天空。
<br/>　　天地至理，在这一瞬，轻松堪破。
<br/>　　萧逸收回了目光，看向周遭。
<br/>　　这一瞬，他仿佛看到了整个地域。
<br/>　　整个地域，空白一片，唯独一团团黑气，分部四周。
<br/>　　黑气，与邪修身上散发的气息，如出一辙。
<br/>　　也就是说，这些，便是分布在这个地域各处的邪修。
<br/>　　这一刻，偌大个地域，一个个邪修的踪迹，悉数出现在萧逸眼中。
