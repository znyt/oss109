　　秦踏天说着，便要动手，不过却被秦月娥一把拦住，
<br/>　　“爷爷，天歌是我丈夫，你不能杀他！我其实只想杀了那个野种！”
<br/>　　叶天歌当即冷冷道：
<br/>　　“你若是想杀我儿子，还是先杀了我吧！”
<br/>　　秦月娥顿时又气又恨，偏偏又无可奈何。
<br/>　　就在这时，林婉容忽然开口了，
<br/>　　“秦小姐，只要你愿意放过我和我儿子，我会立即离开京都，并且今生今世再也不和叶天歌相见，永远不打扰你们的夫妻生活，你看这样，可好？”
<br/>　　听到这话，叶天歌顿时急了，
<br/>　　“容容！”
<br/>　　林婉容眼眸中闪过一抹复杂之色，忽然冷笑道：
<br/>　　“叶天歌，我实话告诉你吧！我当初之所以和你在一起，其实就是贪图你的身份地位，没想到你这么没用，根本给不了我想要的东西，既然如此，那我们还是分开的好，今后你走你的阳关道，我过我的独木桥，再无任何瓜葛！”
<br/>　　林婉容说完这话，搂着怀里的叶尘，将俏脸扭到一旁。
<br/>　　蹬蹬！
<br/>　　叶天歌听到林婉容这话，顿时全身一震，不由得倒退了两步。
<br/>　　以他的头脑智慧，又岂会不明白，林婉容是故意这么说的，但听到心爱的人说出这种话，还是感觉心痛无比，整个人仿佛瞬间衰老了十多岁，颓然道：
<br/>　　“是呀！我的确是很没用！”
<br/>　　一旁的秦月娥，则面露喜色，似乎生怕林婉容反悔，当即道：
<br/>　　“好！这可是你自己说的，那你现在就带着这个野种，给我滚的远远的，永远都不要再回来，倘若让我知道，你们之间有任何来往，我绝对不会放过你们！”
<br/>　　林婉容点了点头，一言不发的抱起叶尘，然后头也不回的朝外面走去。
<br/>　　“容容！”
<br/>　　叶天歌还想去追，却被秦踏天伸手拦住，冷冷道：
<br/>　　“叶家小子，我看在我宝贝孙女的面子上，所以才对你一再忍让，你莫非真当我秦家好欺了？惹怒了老夫，将你叶家灭了又何妨？”
<br/>　　“你！！”
<br/>　　叶天歌怒气冲天，双拳紧握，死死盯着秦踏天。
<br/>　　秦踏天却不以为意，脸上反而露出一抹嘲弄之色，
<br/>　　“怎么？难不成，你还想和老夫动手？哼！以你的年纪，能够达到这等成就，的确算得上极为难得，是不可多得的武道天才，可惜在老夫面前，你什么都不是！”
<br/>　　说完这话，秦踏天身上的气势猛然暴涨！
<br/>　　蹬蹬蹬！
<br/>　　叶天歌在秦踏天强大的气势压迫之下，顿时一连倒退了七八步，方才勉强站稳身形，体内气血翻滚不止，险些喷出一口血来。
<br/>　　“天歌，你没事吧？”
<br/>　　一旁的秦月娥见状，连忙走上前，扶住叶天歌，满脸的紧张之色，并且回头冲着秦踏天嗔道：
<br/>　　“爷爷，你不可伤我夫君！”
<br/>　　叶天歌根本不领情，身体微微一震，将秦月娥的手甩开，再次望了一眼屋外，已经渐行渐远的叶尘母子，并没有再继续追出去。
<br/>　　因为他知道，事情已经成为定局，他此时若是追出去的话，恐怕不但他要死，叶尘母子要死，叶家、林家，全都要跟着倒霉！
<br/>　　秦家老祖秦踏天，不愧是人仙级别的强者，背后又有圣心阁作为依仗，远远不是他所能抵抗的了的，唯有忍耐。
<br/>　　……
<br/>　　寒冬腊月，大雪纷飞！
<br/>　　叶尘躺在母亲的怀抱之中，虽然感觉有些冷，但是心中却无比温暖。
<br/>　　突然，
<br/>　　滴答！滴答！滴答！
<br/>　　一连串的水滴，打在了他的脸颊之上。
<br/>　　叶尘微微抬头，看到母亲此刻的脸庞之上，早已经泪流满面，心里顿时狠狠一抽，一股难以形容的难受感觉涌上心头。
<br/>　　正所谓，母子连心，叶尘完全可以体会到，此时自己母亲心头的痛苦，是何等的强烈！
<br/>　　“不管眼前的这一切，是真实存在的时空，又或者只是一场幻境，既然我来了，那就绝对不能让母亲遭受如此大的羞辱！”
<br/>　　“秦月娥！秦踏天！虽然在外面的世界，我已经杀了你们，但是在这里，你们一样还是要死！”
<br/>　　在心中暗暗发誓之后，叶尘用力抬起自己，那还是婴儿状态的幼小手臂，轻轻拭了拭母亲脸颊上的泪水。
<br/>　　林婉容这才警觉，低头望了一眼怀里的儿子，美眸中顿时露出一抹惊愕之色，泪水也不由得止住了。
<br/>　　她发现自己的这个儿子，似乎和平常的小孩不太一样，不但会帮她擦拭泪水，眼眸也直勾勾的望着她，似乎会说话一般，更像是在安慰她？
<br/>　　林婉容又惊又奇，眼前神奇的一幕，顿时大大冲淡了她心头的痛苦，不由得破涕为笑，在叶尘的额头深深吻了一口，轻声自言自语道：
<br/>　　“孩子，妈妈只要还有你，就已经心满意足了，你可要好好的哦！”
<br/>　　叶尘咿咿呀呀了一会，发现喉咙还是不能发出声音，只好点了点头。
<br/>　　林婉容见状，顿时美眸一瞪，脸上的惊奇之色更浓，这要是换做旁人，估计都会以为，眼前这个婴儿是个怪物，不过毕竟是自己的孩子，林婉容惊愕过后，反而心中欢喜无限，将叶尘搂的更紧了。
<br/>　　半个小时之后，
<br/>　　林婉容带着叶尘，已经登上了返回云州的火车。
<br/>　　而叶尘也没有闲着，经过一番尝试之后，他惊奇的发现，眼前的这个世界，几乎和地球上的那个世界，完全一模一样！
<br/>　　最重要的是，之前那四十八世轮回世界中，都是只有一种法则，譬如造化世界中，就只有造化规则，毁灭大陆上也只有毁灭规则；
<br/>　　但是在这个世界里，前面四十八种规则，居然全都存在！
<br/>　　只不过，这四十八种规则，并非彼此独立，而是完美融合在一起的！
<br/>　　“我明白了！时空法则，是一切事物的载体，同样也是其他法则的容器，所以想要完全参悟时空法则，必须要将其他法则全都融入到时空法则里面，如此才算得上真正的领悟！”
<br/>　　明白了其中关窍，叶尘顿时觉得，这最后一层世界的考验，简直比之前四十八层世界，都要困难的多！
<br/>　　不过，对于刚刚得到母爱的叶尘来说，他倒也并不着急离开这个世界。
