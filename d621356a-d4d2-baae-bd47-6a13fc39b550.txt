　　第668章
<br/>　　程大雷在紧急备战，于此同时，宋伯康也没有闲着，也在为即将到来的两军对垒做准备。
<br/>　　以凉州城为主，在金城、武威、浩甲三座城池间，兵马、粮草、武器来回调动。已派人前往京州请援，看能否相府得到一些支援。
<br/>　　宋伯康已经铁了心要打这一仗，任何能用到的力量都会用到，其目的就是将程大雷的势力从凉州彻底铲除。
<br/>　　山雨未来，却已经是风满楼。便是城中的普通居民，也能感受到如今形势紧张。一面是名正言顺的凉州王，一面是恶名远播的程大雷，众人也在推测着这一战的胜败。
<br/>　　宋伯康这一日在兵营中巡营，检点着手下士兵。事实上，他是一个文官，并不擅长带兵打仗。可他同样也明白，在如今这世道，所谓财富、权力都是虚幻的，只有握在手中的兵才是实实在在的东西。
<br/>　　但这个道理，他明白，相府那位德高望重的老人同样也明白。宋伯康手下握着兵权的戴白，就是那位老人的死忠。假若有一天宋伯康不甘心做相府的马仔，那他能动用的兵力其实并没有多少。
<br/>　　每想到这一点，宋伯康就有些无奈，兵权被别人握着的感觉实在不爽。所以他一直试图培养自己的亲信，收效不大，却一直在做。
<br/>　　或许，等自己除掉程大雷后，方才能真正放开手脚。
<br/>　　想到这一点，宋伯康忽然觉得雄心万丈。
<br/>　　“王上……”一名小校急急忙忙跑过来，在宋伯康耳边低语几句。
<br/>　　宋伯康的表情变得不好看，他挥挥手：“走，去看看。”
<br/>　　戴白和陆洛几人随在宋伯康身后，向事发地点赶去。出事的地方并不在城内，而是在城外一座镇子里。
<br/>　　当宋伯康几人赶到的时候，见里三层外三层已经围满了人。兵丁分开人群，宋伯康走到当中，看到了面前发生的事。
<br/>　　只见在中间一片空地上，一尊石人像破土而出，在石人像后面还刻着六个字：
<br/>　　石人现，蛤蟆王。
<br/>　　宋伯康脸黑得像锅底仿佛，他看见了这一幕，周围的兵丁和人群都也同样看见了。
<br/>　　就算宋伯康就在眼前，他们也已经开始议论纷纷。这蛤蟆二字，明显说的是蛤蟆城那位，那这六个字代表的含义就太明显了。
<br/>　　也太过吓人了。
<br/>　　“老李，这是怎么会事？”一个刚睡醒的老人，还不知发生的事，只是看到人群聚集才凑过来。
<br/>　　“嘿，你还不知道吧，这石人是从土里冒出来的，这是老天给我们的启示呐，蛤蟆王，蛤蟆王……蛤蟆才能是王……”
<br/>　　“假的吧？”
<br/>　　“什么假的，能从土中长出来，这只有神仙才能做到啊。”
<br/>　　这等言论时不时在耳边想起，宋伯康心情糟糕到极点。但同时，他也想不明白，这世人如何能从地上长出来。
<br/>　　莫非真的是所谓的天启？
<br/>　　宋伯康本身是不信的，但难免有些含糊。他这凉州王都含糊了，更何况是周围的兵丁和普通人。
<br/>　　“来人快将这里封了，不许任何人靠近。”紧急状况下，戴白率先打破僵局。
<br/>　　兵丁用大棒将围观的人群驱散，里三层外三层围成一圈。
<br/>　　陆洛悄悄靠近，宋伯康糟糕的脸色令他不敢大声说话，只是小心翼翼问：“王上，这该如何是好？”
<br/>　　那石人面无表情，却像是对自己发出无声的嘲笑，高高在上，神情可恶，像极了蛤蟆城哪位。
<br/>　　“来人，给我将它挖出来。”宋伯康咬着牙道。
<br/>　　诸位兵丁战战兢兢，不敢上前，这可是神启呐，触犯神灵的代价大家可承受不起。
<br/>　　“怎么，你们都不怕死么？”宋伯康冷森森的问。
<br/>　　大家这才恍然大悟，神灵纵然可怕，但能随时夺走自己性命的可是眼前这位。
<br/>　　撑起巨大的勇气，大家忐忑的靠前，心中暗暗祷告：上天有灵，要降责罚就责罚宋伯康，这件事和我们无关呐。
<br/>　　宋伯康紧皱着眉头，一动不动的盯着石人，他心情同样很忐忑。
<br/>　　万一是真的呢？
<br/>　　随着铁锹一下一下挖进去，见见露出这石人的真实模样。事情说破也十分简单，无非将石人事先埋入土中，在下面填满豆芽，灌满水后，豆芽发开会将石人顶出地面。
<br/>　　“雕虫小技。”宋伯康不屑的冷很一声，完全忘记了刚才的紧张：“让人都过来看看，瞧瞧这等鬼把戏，莫让人给糊弄了。”
<br/>　　“是是。”
<br/>　　宋伯康返回凉州城，心中还有些不是滋味。他立刻着手调查这件事，看最近有什么事情发生。不问不知道，一问吓一跳，原来这石人并不是孤立事件，而是无数桩事中微不足道的一件。
<br/>　　某个鱼贩破开鱼腹后，发现里面有一张黄卷，上面写着：蛤蟆王。
<br/>　　几个晚归的旅人在山神庙寄宿，忽然听到山神爷和山神奶奶的聊天。
<br/>　　这宋伯康不是个东西呐，真正该做主凉州的应该是……
<br/>　　街上已经冒出童谣，类似于『宋伯康王八蛋』之类的。
<br/>　　谣言就这样突然冒了出来，在普通士兵间，在平民间已经穿得沸沸扬扬。这等谣言，自然就是程大雷搞出来的。他手下不缺擅长装神弄鬼的江湖术士，现在程大雷还没想到获胜的法子，也不期待这样的行为收到怎样的效果。但我办不了你，恶心恶心你也是可以的么。
<br/>　　都说谣言止于智者，但这世间没有智者。大家本就爱听这怪力乱神的东西，茶余饭后，免不得就说起这些事。一部分人是他本身就相信，另外一部分人是他愿意相信。
<br/>　　在宋伯康的制下，大家也未必有太好的生活。有些人的日子已经过得糟糕得不能再糟糕了，他们渴望改变，所以也相信改变。
<br/>　　宋伯康现在还意识不到这件事将会带来怎样的恶果，如此也符合程大雷的一向的战略：攻心为上。
<br/>　　他只是很愤怒，很恶心，这样统统转变成对程大雷的恼恨。
<br/>　　将手下叫到身边，令他们严查，禁止谣言的传播，刨根问底，看谣言是从那里传出来的。
<br/>(本章完)
