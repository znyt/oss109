　　第272章
<br/>　　这些人都是从青牛山杀出来的弟兄，个人战斗力，互相之间的配合，对于指挥的领会，早已达到一个恐怖的程度。
<br/>　　程大雷的话出来，各人已经开始有条不紊的后退，直到退出院墙外，将战场留给程大雷自己。
<br/>　　“雄黄，给我往死里砸！”徐神机大声喊。
<br/>　　雄黄用树叶包住，每个人都使出吃奶力气，隔着院墙往里面丢。大家都知道程大雷情况危险，都想助他一臂之力，纷纷雄黄在空中散落，院内散开漫天烟雾。
<br/>　　看到这一幕，大家心中都松了口气：这波稳了。
<br/>　　“砸准点！”院内传来程大雷的怒吼：“砸我脑袋了！”
<br/>　　程大雷现在的情况不是危险，是相当危险。聚义厅前的院内，到处都是弥漫开来的雄黄，白蛇王很显然很痛苦，速度变得更快，攻击更有力量……它被激怒了。
<br/>　　这样体量的雄黄肯定会给白蛇王造成一定困扰，但说到给它造成伤害……程大雷自己都觉得有点悬。
<br/>　　他将身无彩凤双飞翼施展到极致，绝不在任何一个位置多停留一秒，可即便如此，程大雷还没寻到任何攻击机会。
<br/>　　这样皮糙肉厚的猛兽，很难说能够一斧头砍死，可如果给了它攻击的机会……程大雷幻想过自己的死法有很多，但绝不包括成为蛇腹的一部分。
<br/>　　他摇身一旋，攀上院内一棵大杨树，再起身，直接落在树梢，这已经是杨树的最顶端，离地三十米。
<br/>　　程大雷长出一口气，真悬呐。
<br/>　　“大当家，你没事吧？”院外的人喊道。
<br/>　　“还行，棒着呢。”程大雷拍着胸脯：“且看本当家今天如何单枪匹马，铁斧屠龙。”
<br/>　　“大当家，小心，树，树倒了。”
<br/>　　程大雷一直盯着白蛇王的动静，此刻怎么会没有发觉，白蛇王缠住杨树，结果它根本没有往上爬，身子用力一扭，只听咔嚓咔嚓声响，水盆粗细的杨树硬生生被折断。
<br/>　　程大雷跃到房顶，再折身，白蛇王缀着他不放，蛇躯扫到之处，房倒屋塌。程大雷就在漫天的烟尘中和巨蛇兜着圈子。
<br/>　　他一直在寻找进攻的机会，手中的大斧变得炙热滚烫，可巨蛇的速度实在太快，力量也太过恐怖。
<br/>　　一张血盆大口向程大雷扑来，从猩红巨口中散发出一阵恶臭，程大雷脑袋一晕，险些跌倒在地。
<br/>　　亏得求生的本能刺激了他的潜力，他使出轻身功夫，纵身跃起，已经出现在院外。
<br/>　　“大当家，大当家……”
<br/>　　众人将踉跄的程大雷扶住，程大雷此刻浑身上下已经被湿透，他抹了把额头的冷汗，仍旧心有余悸。
<br/>　　“太可怕了，差点着了道。”
<br/>　　“大当家，你不是说要单枪匹马，铁斧屠龙么？”徐神机。
<br/>　　“快别扯淡了。”程大雷摇摇手：“撤。”
<br/>　　众人退回到登岸的浅滩，白蛇王并没有追来。本来一切都很顺利，却是谁也没想到岛上竟盘踞着一条恐怖的蛇王。
<br/>　　“咱们已经费了这么大力气，真要放弃这座岛？”徐神机。
<br/>　　“我什么时候说放弃这里了。”程大雷还有些头晕，呼吸着周围的新鲜空气：“只是我恰好想到一个法子，觉得没必要拼命了。”
<br/>　　“什么法子？”
<br/>　　“需要一些酒，或者很多。”程大雷道：“看哪里能搞到酒？”
<br/>　　西门春长和他的女伴被从船舱内拎了出来，现在二人已经被吓瘫，毕竟落在这群魔王手中，随时有可能脑袋落地。
<br/>　　“想活命么？”程大雷问：“透漏点消息给我，附近哪里能搞到酒？”
<br/>　　“浩甲城有户人家是酿酒的，只要你们放我回去，明天我肯定把酒送来。”
<br/>　　“太麻烦了。”程大雷道：“你写一封信，让你的家人送酒过来，见到酒后，你就可以活。”
<br/>　　西门春长咬破手指，在自己的袍子上写了一封信，最好脸色苍白的递给程大雷：“交给梅姐，只有她肯救我。”
<br/>　　派出赵子龙和秦蛮去送信，今夜大家还不能登岛，只能在岛屿外围的浅滩上露营。
<br/>　　究竟能不能顺利登岛，还要看明天的『赎金』能否送到。程大雷来到西门春长身边，问：“那梅姐是你什么人呐？”
<br/>　　“是小人的发妻。”
<br/>　　“夫妻感情怎么样？”程大雷热心得像个街道大妈。
<br/>　　西门春长已接近崩溃，他与发妻的感情何止是不好，简直是糟糕头顶，平素他眠花宿柳，自己也不知道与发妻有多久没有同行。可这生死关头，能想到的女人有很多，可能救自己的，还是只有那一个人。
<br/>　　“怎么说呢，我只能说祝你好运。”程大雷道：“当然，也祝我好运吧。”
<br/>　　半夜的时候，秦蛮和赵子龙回来，表示信已经送到。程大雷只好和大家一起等待着，留给他的时间当真所剩无几，假若明天来的不是赎金，而是剿匪的官兵，那大家的防御……或许只能依靠聚义厅中的白蛇王。
<br/>　　次日黎明，东方破晓，程大雷忽然看到远处水面出现船帆。他忙叫醒众人，诸人严阵以待，随时准备战斗。
<br/>　　“船头好像是一个妇人？”赵子龙目力好，第一时间发现。
<br/>　　“春哥，春哥……”
<br/>　　那妇人一路叫着春就来了，在靠岸时踉跄将西门春长抱住，急得眼泪已落下来。
<br/>　　程大雷看着这位被唤做梅姐的妇人，也无怪西门春长冷落她，此人满脸麻子，腰有水桶粗细，让人见之生厌。
<br/>　　“诸位大王，你们要的酒都在船上，求求诸位，放我夫君一条性命。”
<br/>　　“放人，卸货。”程大雷也长出了口气。
<br/>　　诸人将船上的酒卸下来，将西门长春放走，等忙碌完这一切，太阳已经从东方出来，浅滩上堆满酒坛。
<br/>　　“当家的，你是要用这酒把蛇王烧死么？”徐神机。
<br/>　　“烧什么烧，我是要灌醉它。”
<br/>　　“灌醉？”徐神机道：“是要给蛇王献酒投降，它是一条蛇，吃着一套么？”
<br/>　　程大雷不是很想搭理他，命令诸人一人搬一坛酒上山。
<br/>　　昨夜那蛇王并未追出来，此刻依旧盘踞在聚义厅中，蛇躯像一座小山。
