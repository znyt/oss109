/>　　<br/>　　其他三人也纷纷开口道：<br/>　　<br/>　　“是啊！叔叔们没有恶意，只是来取一件东西而已！”<br/>　　<br/>　　“我们真的都是你爸妈的朋友！”<br/>　　<br/>　　“你爸爸叫小林健次郎，你妈妈叫韩英，他们两个都在柳生株式会社工作，我说的没错吧？”<br/>　　<br/>　　小林舞依听到他们真的认识自己的父母，小脸上不由得露出一抹狐疑之色，忍不住望了一眼旁边叶尘，却见他低着头一言不，一时间也顿时没了主意，<br/>　　<br/>　　“你们……真的是我爸爸妈妈的朋友？来我家找什么？”<br/>　　<br/>　　四人见忽悠住了小林舞依，顿时纷纷露出一抹喜色，那中年男子继续装模作样道：<br/>　　<br/>　　“是这样的，舞依酱，你的爸爸妈妈临死之前，正在从事一项伟大的研究，他们的研究笔记对公司乃至对整个国家都十分重要，你看你能不能帮叔叔把它找出来？”<br/>　　<br/>　　小林舞依此时也已经看出来，这些人不怀好意，不过有叶尘在，她倒是一点也不怕，<br/>　　<br/>　　“我爸妈的笔记吗？我的确知道在哪，不过……我凭什么给你们？”<br/>　　<br/>　　四人听到这话，先是一喜，继而纷纷露出了一抹狰狞之色，<br/>　　<br/>　　“不给？这可由不得你了！”<br/>　　<br/>　　四人说着，便要捋起袖子准备走上前来，将小林舞依擒住，然后刑讯逼供。<br/>　　<br/>　　至于小林舞依旁边的叶尘，则直接被他们忽视了。<br/>　　<br/>　　毕竟，不过一个十几岁的少年而已。<br/>　　<br/>　　叶尘不由得摇了摇头，也懒得和他们废话，直接心念一动，<br/>　　<br/>　　“搜魂！”<br/>　　<br/>　　四人顿时瞬间呆立当场，继而一道白光分别没入四人眉心之中。<br/>　　<br/>　　转眼之间，叶尘已经从四人的记忆中找到了自己想要的东西，然后随手弹出四道三昧真火，瞬间将四人焚烧一空，连渣都没有剩下。<br/>　　<br/>　　一旁的小林舞依虽然见识过叶尘的手段，依旧觉得震撼不已。<br/>　　<br/>　　而叶尘在消化了从四人脑海中抓取的信息之后，这才缓缓睁开双眼，转头望向一旁的小林舞依道：<br/>　　<br/>　　“丫头，原来害死你父母的真正元凶，不是那个杉口组的浅野，而是柳生株式会社的社长，柳生正平！等等，这家伙居然是那个柳生正龙的弟弟？”<br/>　　<br/>　　“也罢，那就新仇旧恨，一起解决了吧！”<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
