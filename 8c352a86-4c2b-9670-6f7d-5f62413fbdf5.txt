　　第515章
<br/>　　这老者形容枯槁，长发披散下来，露出一张干枯的脸。整个人如鬼如仙，平白给人一种恐怖的感觉。
<br/>　　他不知在这大雪峰上住了多久，即便附近的牧民，也并不知道这终年酷寒之地，竟还有人居住。
<br/>　　老者无名无姓，也看不出多大年纪，同样也无人知道他的来历。不过，显然他和白衣老尼是认识的，只是一见到对方就迫不及待的跳崖逃走，令事情透着一些奇怪。
<br/>　　见他去而复还，老尼也没有觉得奇怪，道：“我请你出山，帮我杀一个人。”
<br/>　　“什么人呐？”老者上下打量老尼一遍，忽然一惊：“你已经和他交过手了，而且还受伤了。哎呀呀，你都不是他对手，我去帮忙又能做得什么，还不是一块儿被他杀了。”
<br/>　　“莫要看不起自己，当年西蜀巫医的传人，举手投足便能覆灭一城，何况今日只是杀一人。”白衣老尼平静道。
<br/>　　西蜀巫医这四个字，似乎令老者想起一些不好的事情。他立在哪里，干笑两声：“西蜀已经亡了，过去的事情又何必再提。”
<br/>　　事情的确已经过去太久，久到当初活着的人差不多都已经死了。的确是很久很久以前，那个时候天下大乱，还没有如今的帝国，只有十九国互相争斗的乱世。那时候天生狂人李蛮子出世，一杆铁枪扫荡十九国，打下一个铁通般的大武王朝。
<br/>　　当然，那也是一百二十年前的事了，如今天下人想的，是如何从现今的乱世中夺得一二分利益。可世间事，凡走过必留下痕迹，被李蛮子铁枪扫荡过的十九国，总还是有些遗老遗少活着。
<br/>　　一百二十年的时间，说短不短，说长也并长，不足以消磨这些遗老遗少的复国之梦。
<br/>　　他们一代人一代人传下来，怀揣故国旧梦，在如今的帝国生根发芽，如今已经折腾出一片怎样的势力，怕谁也说不清楚。
<br/>　　白衣老尼道：“你住在大雪峰上，距离凉州很近，知道不知道凉州出了一个枭雄，短短几年便折腾出一片不小的势力。”
<br/>　　“谁呀，我怎么没听说过？”老者问。
<br/>　　“大雪峰距离琴川并不远，你不该没听过他的名字。”白衣老尼皱起眉头：“你这些年都是怎么过的？”
<br/>　　“就是很平常啊，每天打两只小兽，焚雪为水。”老者自豪道：“一直活得还算不错，一个人生活还是很轻松的。”
<br/>　　白衣老尼难以置信的打量着他，道：“那么，你有多久没和人说过话了？”
<br/>　　“这个……呃……”老者皱起眉头：“有十年还是八年了，我没有仔细记过。”
<br/>　　老尼无可奈何的摇摇头：“随我下山吧，如今天下乱了，正是你我做事的机会。”
<br/>　　老者像是被踩到尾巴一样，差点从地上跳起来，口中不断道：“我便知道，我便知道……公主殿下，都过去一百年了，世上那还有什么西蜀东楚，不过就是一群过街老鼠而已，你还真以为我们能够复国。”
<br/>　　白衣老尼的表情突然顿住，似乎是很久没有听到『公主』这个称呼。这老者是当年蜀国巫医的传人，蜀国巫医本就天下扬名，独居这么多年潜心研究，真不知他的医术如今在什么程度。而这白衣老尼，算是西蜀的皇族，如果蜀国还在，当有一个公主的身份，可惜蜀国早在一百年前就亡了。
<br/>　　从她出生开始，上一代人就在她心里扎下复国的念头。这么多年都是抱着这个念头活着，但其实也不知要如何复国。
<br/>　　后来侥幸得到聂隐娘这样一个习武天才，年纪轻轻就成了天下超一流此刻。于是便命令聂隐娘去杀一些蜀国叛臣的后人。其实究竟杀了这些人和复国有多大关系，她也没有仔细想过。
<br/>　　如今，老者一句话问完，见老尼表情犹豫，便大概能猜出来她心中想着什么。
<br/>　　“公主要杀的人是谁啊？”
<br/>　　“他叫程大雷……”
<br/>　　“名字听上去也平平无奇。”老者道：“有什么来历，一定要杀他？”
<br/>　　于是，白衣老尼便将程大雷所做的事情说了，其实她知道得也不清楚，无非都是道听途说，等江湖上的传言传入她的耳中，还不知程大雷的变成什么模样。
<br/>　　“原来出身是个山贼呐。”老者叹了一声：“如此说来，他是帝国的仇人呐，公主如果想复国，不应该拉拢他么，怎么又要杀他呢？”
<br/>　　“这……”白衣老尼顿住。
<br/>　　老者悠悠叹了口气，他便知道这老尼的脑子不是很好使。毕竟，能将复国的念头一直揣到现在，脑子也清楚不到哪里去。
<br/>　　“我们都已经老了。”老者盘腿在地上坐下，道：“我年轻时，也是风姿潇洒的俊品人物。琴棋书画，医卜星相无所不通，多少千金小姐憋着心思想和我说一句话。后来怎样呢，东家要拉我入伙，西家要拉我入伙，帝国又要杀我。我的父母妻儿，都在这个过程中惨死，我便躲到了这大雪峰上，如此一躲就是三十年。有时想想，如此一生过去，实在辜负了一身所学。可后来渐渐这心思就淡了，毕竟我已这般年纪，或者那天没醒过来，就埋骨雪山了。”
<br/>　　“复国……”老者揉揉眼睛：“复个屁国呀。”
<br/>　　白衣老尼沉默听他说完，道：“我的徒儿死了。”
<br/>　　“死了便死了么，谁还没个死，你我又能活多久。”老者道：“依我看，你也莫下山了。就在大雪峰上住下，我打猎你烧水，日后我们无关谁先死，身边总还有个人守着。”
<br/>　　白衣老尼以不可思议的目光盯着他，道：“几十年了，当年风姿无双的巫仙孟玄青，怎么变得如村夫野汉一般。”
<br/>　　名叫孟玄青的老者笑了笑，笑容有些威胁：“你要是几十年不和人说话，看见一个人，终也不舍得她走的。公主，我已经老了，老胳膊老腿，真没办法陪你下山了。”
<br/>　　白衣老尼沉默半晌，悠悠叹了口气，道：“你说得未必没有道理，可人活着……总是要做些事情的。”
<br/>(本章完)
