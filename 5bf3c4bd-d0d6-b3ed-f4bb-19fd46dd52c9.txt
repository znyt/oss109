　　第322章
<br/>　　兵临城下，旌旗蔽日，帅字旗高悬，上书斗大一个莫字。
<br/>　　程大雷站在瞭望台上，透过单筒望远镜看到这一幕。莫鸣米在陆上扎营，距离蛤蟆寨大约五里。根据对方的帐篷和旗帜判断，人数当在万人以上。
<br/>　　已不是第一次面对这种状况，程大雷没有初次的手忙脚乱。他在欣赏对方大营时，还不忘指点一下手下的指挥官们。
<br/>　　“大家看他们扎营的位置，临岸解决了水源问题，高地扎营，视野开阔，一旦有人袭营，第一时间就可发现。”程大雷语带赞许道：“这莫鸣米是个行家呐。”
<br/>　　“大当家，我们该怎么守？”徐神机。
<br/>　　“啊……这么嘛，其实我还没有想好。”
<br/>　　程大雷目前的确还没有清晰的战术。目前己方最大的劣势就是人数太少，双方兵力比几乎达到十比一。虽然守城用不了太多人，以少胜多的事情程大雷也不是没有做过。但琴川和青牛山的地形不同，山地本就不适合大部队同时推进，当时只要守着山谷城门，就能立于不败之地。
<br/>　　但现在岛上并没有修建高墙，本来是打算用周围的滔滔河水阻止大兵，可程大雷没想到，河道竟然结冰了。
<br/>　　目前程大雷也不了解莫鸣米打算如何进攻，能做的也就是静观其变，看对方如何来攻，自己再决定如何去守。
<br/>　　此刻蛤蟆寨已经进入战斗状态，刀出鞘，弓上弦。只等大军来袭，便释放出该有的杀机。
<br/>　　程大雷很满意现在的士气，即便面对超过十倍的敌人，弟兄们也只是紧张，而没有畏惧。如此说来，连日来的训练也不是无用功。
<br/>　　他透过单筒望远镜，看着对面莫鸣米大营的动静。昨夜已扎好大营，到现在还没攻过来是为什么事。
<br/>　　只见营地中央用土垒起一座点将台，有人擂鼓，莫鸣米站在点将台上，正说着些什么。
<br/>　　这应该是战前动员了吧。
<br/>　　这时代的战争，出兵之前其实很麻烦的。要祭天，祭地，祭苍生，说一说对方如何罪大恶极，己方又是如何替天行道。
<br/>　　然后，程大雷就看到有人被拉到点将台上，莫鸣米亲自挥刀，斩下了对方的脑袋。
<br/>　　“呃……这是做什么？”
<br/>　　“杀人祭旗么。”徐神机理所当然道：“这都是老规矩了，估计被杀的是个死囚犯之类的吧。”
<br/>　　“快，让大家准备防御，估计要打过来了。”程大雷立刻道。
<br/>　　程大雷紧盯着对面大营，只见有一黑马从辕门出来，马背上坐着一人，看方向是向这边来的。
<br/>　　“怎么会事，派一人就敢来打我们，太不把我蛤蟆大王放在眼里了吧。”程大雷。
<br/>　　“好像不是来打我们的……”徐神机也面露困惑。
<br/>　　只见那人骑着黑马缓缓靠近蛤蟆寨，在接近瞭望台时停下。
<br/>　　“呔，蛤蟆寨的贼人停着，我是琴川守莫将军的人，让程大雷上前答话。”
<br/>　　程大雷根本没有搭理他，是徐神机冲下面道：“我家寨主忙得很，没功夫见你，有什么话对我说也是一样。”
<br/>　　那人整了整身上袍子，忽地抽弓搭箭，将一封信射到瞭望台上。
<br/>　　程大雷和徐神机凑在一起，将信打开。
<br/>　　“琴川一地，人杰地灵，灵山秀水，福报绵长……”
<br/>　　台下那人朗声大喊，与信上内容一模一样。
<br/>　　程大雷此刻才明白，这是一封战书，大概内容是程大雷如何为非作歹，作恶多端，什么三岁偷看人洗澡，八岁强暴人姑娘的事都提到了。又说莫鸣米如何宽宏大量，一再忍让，最后实在看不得畜生活在人间，才派兵来伐。当然，上天有好生之德，只要大家将程大雷交出来，举手投降，莫将军就可以既往不咎。
<br/>　　“一派胡言！”徐神机愤怒道：“他们怎么可以这样诋毁大当家。”
<br/>　　“好啦，好啦。现在生气有什么用，如今人家已经划下道来，如何迎敌，就全拜托徐军师了。”程大雷道：“他们下了战书，你就干脆写一遍气壮山河的檄文，吓破他们的狗胆。”
<br/>　　“这个……那个……”徐神机结结巴巴。
<br/>　　“好啦，好啦。”程大雷也知道凭徐神机肚子里那点墨水，描黑一枚围棋都困难，更何况是写什么檄文。他啪地一声拍案而起，冲下面道：“呔，你家程爷爷在此，告诉姓莫的，你家程爷爷既然来了，这琴川一地，就不是他说了算的，要打就打，休要废话。”
<br/>　　替莫鸣米传战书之人听到这话后，拨马要走，程大雷盯着其背影，道：“来人，给我留一留他。”
<br/>　　赵子龙立在箭楼上，听到这话后抽弓搭箭，只听嗖地一声，箭射在黑马身上。黑马唿律律一声长嘶，一下将马背上那人甩在地上。
<br/>　　蛤蟆寨众弟兄看到这一幕，发出一阵大笑，那人惊慌失措的爬起，在冰面上连滚带爬，却连头都不敢回。
<br/>　　这人一路逃回大营，将消息禀报给莫鸣米。莫鸣米知道后也不生气，毕竟他对这一切早已预料到。
<br/>　　莫鸣米大手一挥：“出兵。”
<br/>　　程大雷等在蛤蟆寨，觉得这下应该打来了吧，然而事实上，莫鸣米说了出兵，也没有立刻进攻。
<br/>　　程大雷只看到对面大营人头攒动，就是不见有人攻来。程大雷等得耐心都快消耗尽了，这该不会是莫鸣米的计策吧。
<br/>　　直到第二天午时，才看到对面大军出营，排成整整齐齐的方阵，向这边奔来。
<br/>　　程大雷松口气，该来的终究要来了。
<br/>　　不得不说，帝国的军事组织能力低下，从扎营到发兵，三四天时间都已经过去了，结果这才打过来。你说你莫鸣米原先是个山贼，就好好做你山贼这份有前途的工作不好么，为何也要学着向帝国军方这样腐朽落后。
<br/>　　既然对方已经攻来了，自己能做的，也就是兵来将挡，水来土掩。目前山寨不过一千多号人，程大雷其实可以做到如臂使指，但他刻意下方了一些权力。让秦蛮，赵子龙，关鱼，张肥，高飞豹各主持一面岛屿的防守。
<br/>　　
<br/>(本章完)
