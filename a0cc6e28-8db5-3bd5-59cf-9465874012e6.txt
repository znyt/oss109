　　第348章
<br/>　　春寒料峭，南燕北飞，正是万物萌苏的时节。程大雷伫立河边，捂着裤裆，竟无语凝噎。
<br/>　　虽说已经入春，但天气依旧很冷，风从西北而来，浩浩荡荡吹过荒原，用刀子般的小手撩拨程大雷的躯体。
<br/>　　春姑娘一来，程大雷就把裤子脱了。
<br/>　　“究竟哪个王八蛋偷了老子的牛！”
<br/>　　墨玉乌牛乃是系统所赠，顶级坐骑，附带技能：野蛮冲撞，石力。唯一的缺点是长得丑，但能打能跑，程大雷近几天使用下来可以说得心应手。
<br/>　　这黑牛通人性，程大雷也不去管它，就放它在河边吃草。可谁知从水里爬出来，黑牛竟然消失不见。
<br/>　　丢不丢还在其次，关键是如今的自己怎么回去。好歹自己现在也是琴川关的首领，如果传出去自己是个暴露狂……难道自己就不要面子么。
<br/>　　可这天高地阔，去哪里寻那黑牛，总不能熬到黑夜，摸黑回去吧。
<br/>　　程大雷拎着斧头，鬼鬼祟祟走在荒原上，心里暗暗发狠。
<br/>　　“若真是谁把老子的牛偷走，老子非一斧一个，活活劈了他。”
<br/>　　一路胆战心惊，生怕被别人发现，走了不远，就听到前面有说话声。程大雷不敢再靠近，窝在草丛里往远处瞧。
<br/>　　对方人数还不少，为首几匹骏马，马上人裹着精致的锦袍，其中还有个粉雕玉砌的小姑娘。
<br/>　　而自己那头乌牛，恰被他们围在其中。
<br/>　　这畜生怎么逃到这里来了，自己的衣服现在还在牛背上。
<br/>　　虽然发现了，但程大雷还不敢露面，身上没有穿衣服，毕竟是有些心虚。
<br/>　　一众人围着黑牛，脸上有说有笑。
<br/>　　“这黑牛看上去好奇怪，丑虽然丑了些，但看着蛮有趣。”粉脸姑娘道。
<br/>　　“婷姑娘假若喜欢，这黑牛便是婷姑娘的，待我擒了这黑牛，送给婷姑娘。”一个蓝袍男人道。
<br/>　　“哈，若是送，也自然是宋公子送，你送的，怕是婷姑娘不会收。”一个书生儒衫的男人道。
<br/>　　粉脸姑娘脸上略红了红，却偷偷抬眼看向一名紫袍公子，若有期许。
<br/>　　紫袍公子姓宋，名游渠，乃是宋伯康的公子。这次宋伯康被相府派过来主持西北之事，他是领人过来看看，一为打探情报，二为游山玩水。
<br/>　　其他几人都是他的玩伴或门客，那粉脸姑娘来头可不小。乃是如今帝国丞相的孙女，名唤薛问婷。薛家和宋家都有意促成这二人的亲事，算是政治联姻。
<br/>　　蓝袍男人名叫钟武才，他眼珠一转，目光有些狡黠。
<br/>　　“我擒住这黑牛，自然也是替宋兄送，呔，现在就让我擒住这黑牛，给公子与小姐献宝。”
<br/>　　看上去很有冲劲的钟武才，其实有几个心眼，他出身不高，攀上宋家的高枝，整个家族的地位也是水涨船高。对于薛宋二人自然不敢得罪，当然，还要谄媚得不动声色。
<br/>　　他纵马扑向黑牛，身体从马上跃起，向黑牛扑来。
<br/>　　黑牛抬起头，眼中似乎有些困惑，如果它通人性，想的一定是：这货在干什么？
<br/>　　就连关鱼和秦蛮都摸不着黑牛，更何况是钟武才。本来安静的黑牛，突然提速，钟武才生扑在地上，结结实实砸了个狗啃泥。
<br/>　　如此，换来同行人的哄堂大笑。
<br/>　　钟武才爬起来，脸上却没有半点尴尬，只是道：“宋兄，这牛有些奇怪，我是擒不住他，你可要小心些。”
<br/>　　既然是绿叶，就要好好做一片绿叶，用自己的无能显示宋游渠的本事。如果你把红花的风头抢了，他会待见你么。
<br/>　　宋游渠有些傲慢，经常板着一张冷脸。
<br/>　　“无非是头畜生而已，还能被它逃了。”说着，他解下身上的披风，丢给身后同伴。
<br/>　　“宋哥哥，帮我擒住它，我要养它做宠物。”薛问婷拍手笑道。
<br/>　　宋游渠抖索精神，手里抓着一套索，一看，他便比钟武才有些驯兽的手段。当然，钟武才未必不知道，只是假装不知道而已。
<br/>　　黑牛原先在河边吃草，吃着吃着就走远了，身边围着这么多人它也不太关心。
<br/>　　顶级坐骑，已经暗通人性，它当然没有智慧知道这些人想做啥，可隐隐已经感受到这些人的敌意。
<br/>　　常常用牛脾气来形容一个人的性格，简单来说就是：牛的脾气不是特别好。
<br/>　　它从安静吃草的家牛，转变成战斗形态，后蹄刨地，身上长长的毛发抖动起来。
<br/>　　凶相必现。
<br/>　　刚才薛问婷说这黑牛有些丑，这只是墨玉乌月牛的一个特点而已，另外一个特点是：凶。
<br/>　　又凶又丑。
<br/>　　宋游渠还没有出手，黑牛就率先发动攻击，野蛮冲撞技能发动，速度很快，力量很大，直接在队伍中趟出一条道路，人仰马翻。
<br/>　　来回走这几趟，队伍中就没有一个人能站着，黑牛得意的打个响鼻，有些嚣张。
<br/>　　宋游渠一伙跌在地上，干净的衣裳沾满黑泥，泥里面还有马粪什么的。
<br/>　　都是十指不沾阳春水的人物，哪里受过这个。
<br/>　　薛问婷气得俏脸煞白，怒道：“杀了它，我要宰了它！”
<br/>　　众人翻身上马，都亮出兵器，黑牛有些猫耍耗子的心思，在队伍中乱闯，时不时掀翻一人。
<br/>　　忽地，它似乎注意到草丛中的程大雷，怒意消退，打着响鼻冲了过来。
<br/>　　“想逃，门都没有！”薛问婷怒喝，抓着一把细剑，追向黑牛。
<br/>　　约跑出二三十米远，感觉这黑牛跑得越来越慢，她眼中绽放着杀机，要杀了这黑牛泄愤。
<br/>　　说时迟那时快。
<br/>　　前方草窝中，突然一道人影跃起，赤果身子，只有腰间一条犊鼻裤。
<br/>　　哇！
<br/>　　薛问婷发出一声大叫，赶紧捂住了双眼。
<br/>　　等她再回过神来，看牛背上坐着一条赤果丑汉，精赤着胸膛，一手扶着牛头，一手托着一柄大斧，呵呵傻笑。
<br/>　　这时代的确封建些，沾衣裸袖便为失节，更何况对方还这么坦荡的出现在自己面前。
<br/>　　由羞转怒，由怒转恨，薛问婷眉间一厉，持剑扑向程大雷。
<br/>　　“腌臜贼，竟敢调戏本姑娘，去死吧！”
<br/>　　
<br/>(本章完)
